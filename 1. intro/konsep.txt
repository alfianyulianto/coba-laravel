"Layouts Using Template Inheritance"
@extends('folder.name') ; untuk mendefinisikan tampilkan mana yang akan diwariskan ke tampian anak
@section('name') : untuk mendefinisikan bagian kontent
@section('name', 'value') : untuk mendefinisikan bagian kontent, yang mana parameter kedua dapat dijadikan sebuah value yang dapat digunakan
@yield('name') : untuk menampilkan kontent dari sebuah section
$yield('name', 'value') : yield directive juga menerima sebuah default value sebagai parameter. Default value akan digunakan ketika sebuah yield undefined
